       7              : This function implements a protein inference algorithm for shotgun proteomics using:
       8              : 1. Connected component analysis to partition the problem into independent protein-peptide clusters

      12              : The algorithm handles the following cases (based on Nesvizhskii & Aebersold, 2005):
      13              : 
      14              : - Case A: Distinct proteins (proteins with non-overlapping peptide sets)
      15              : - Case B: Differentiable proteins (proteins with shared peptides but also unique peptides)
      16              : - Case C: Indistinguishable proteins (proteins with identical peptide sets)
      17              : - Case D: Subset proteins (when one protein's peptides are a subset of another's)
      18              : - Case E: Subsumable proteins (when multiple proteins together can explain all peptides)
      19              : - Case F: Protein groups with shared peptides only (no unique peptides for any protein)
      20              : - Case G: Complex interconnected proteins (with various shared peptide patterns)
      21              : - Case H: Concatenation of cases A-G
      22              :

      25              :   multiple protein identifiers separated by semicolons in the name field, indicating that the peptide

      27              : - `peptides::Vector{PeptideKey}`: Peptide identifiers corresponding to the proteins vector.
